

# E-Commerce Sales Analysis Report

## 1. Project Overview

This project performs a **full data analysis pipeline** on e-commerce sales data. The main goals are:

* Understand sales trends over time
* Identify top-performing products
* Analyze regional performance
* Generate meaningful visualizations to support insights

The analysis uses Python libraries **Pandas** and **Matplotlib**, and produces **charts for trends, comparisons, and distributions**.

---

## 2. Dataset Description

The dataset `sales_data.csv` has **100 rows** and **7 columns**:

| Column      | Description                |
| ----------- | -------------------------- |
| Date        | Date of the sale           |
| Product     | Product name               |
| Quantity    | Units sold                 |
| Price       | Price per unit             |
| Customer_ID | Unique customer identifier |
| Region      | Region of the customer     |
| Total_Sales | Total revenue for the sale |

> Data was cleaned to remove missing values and convert types correctly.

## 7. Technical Details

* **Language:** Python 3.x

* **Libraries:** Pandas, Matplotlib, OS

* **Analysis Steps:**

  1. Load dataset from CSV
  2. Validate required columns
  3. Clean data and remove missing values
  4. Aggregate sales by month, product, and region
  5. Generate 5 visualizations (line, bar, pie charts)
  6. Save charts in `visualizations/` folder and print summary metrics
